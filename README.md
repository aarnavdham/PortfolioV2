# NVISION — Deploy to Vercel

## Quick Deploy
1. Go to vercel.com
2. Sign up with GitHub
3. Click "Add New" → "Project"
4. Import your GitHub repo (upload all these files to it first)
5. Click "Deploy"
6. Done — your site is live with all animations working

## Local Development
```bash
npm install
npm run dev
```
Open http://localhost:3000
